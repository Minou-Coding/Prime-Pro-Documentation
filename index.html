Fonctionnalités Clés

Modulaire

Tout est optionnel. Activez uniquement ce dont vous avez besoin pour garder votre serveur léger et rapide.

Sécurisé

Système anti-raid, vérification complète et isolation stricte des configurations entre les serveurs.

Démarrage Rapide

Déployez Prime Pro+ en moins de 2 minutes chrono.

1. Invitation Admin

Pour garantir que tous les modules fonctionnent sans erreur (création de salon, gestion des rôles), le bot requiert les droits d'Administrateur.

Cliquez sur ce bouton sécurisé pour lancer l'invitation avec les droits pré-configurés.

Ajouter au Serveur

2. Premiers Pas

Une fois le bot sur votre serveur, il est silencieux par défaut. Réveillez-le avec ces commandes :

  • /setup : Lance l'installation de base (Vérification).
  • /setup-tickets : Installe le support ticket.

Système de Vérification

Le bouclier anti-raid ultime pour votre serveur.

Commandes

SETUP /setup

La commande magique. Elle configure :

  • Les rôles Vérifié et Non Vérifié.
  • Le salon #verification.
  • Les permissions des salons pour rendre le serveur privé.
CONFIG /config verification

Modifier les rôles utilisés par le système.

Paramètre Description
unverified-role Rôle donné à l'arrivée.
verified-role Rôle donné après validation.
VISUAL /config images

Changer l'apparence de l'embed de vérification.

Paramètre Description
image URL de l'image principale (bannière).
logo URL de la petite icône (footer).

Rôles & Réactions

Créez des menus interactifs pour vos membres.

Création

ADD /config-roles add-button

Ajoute un nouveau choix de rôle dans votre configuration.

Paramètre Description
role Le rôle à attribuer.
label Texte sur le bouton.
emoji (Optionnel) Emoji sur le bouton.
DEPLOY /setup-panel

Fait apparaître le menu de sélection des rôles dans le salon actuel.

Support & Tickets

Offrez un support client de niveau professionnel.

INSTALL /setup-tickets

Installe le panneau de création de tickets.

SETTINGS /config-tickets

Configurez le comportement des tickets.

Sous-commande Description
set-category Catégorie où s'ouvrent les tickets.
set-logs Salon pour l'historique (transcripts).
add-support Ajouter un rôle Staff autorisé à répondre.

Logs & Accueil

Gérez les arrivées et les départs avec élégance.

Configuration

CORE /config-accueil

Paramètres principaux du module.

Sous-commande Description
set-welcome-channel Salon pour les arrivées (Image + Message).
set-leave-channel Salon pour les départs.
set-images Configuration du fond et du logo.

Note sur les Images : Le bot génère automatiquement une image de bienvenue incluant l'avatar du membre. Utilisez une image de fond de haute qualité (1024x500 recommandé).
